# Web Scraper for extracting malayalam Text
  This is meant for educational purposes only. The text is extracted from 
  https://www.manoramaonline.com/ 
  
  ## Running the scraper 
    scrapy crawl news
   
   news  is used to refer to the name of the web spider . The result is stored in **res.txt**
